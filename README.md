# Sales performance and Hr Data-Analysis (Interactive Dashboard creation using Tableau)
## Project Obejective 
The objective of this project is to analyze and visualize Sales and HR performance data using Tableau. The dashboard integrates key business metrics to provide a comprehensive view of organizational performance.

- <a href="https://github.com/koredecoker1993-cpu/-Data-analysis.-Hr-Sales-Dashboard/blob/main/HR%20Data%20set.csv">Dataset</a>

## Question (KPIS)
For Sales 
- What is the total revenue generated and how many courses/products were sold?
- Which products drive the highest sales volume?
- Which product categories contribute most to revenue and quantity sold?
- Which customer segments (Corporate, Home Office, Consumer) generate the most revenue?
- How does sales distribution vary by location?
- How has year‑over‑year sales performance changed?
